Show the log of your Ghost site (not the CLI) using the log utility command.

To use the command `ghost log` navigate into your Ghost installation folder.

### Arguments

```bash
# Run ghost ls and copy the name of your site
ghost log name

# Follow the log line
--follow, -f
```
